NUST LMS Auto-Feedback:

Requirements:
1. Google Chrome
2. Python 2.7

Please install the above mentioned dependencies before proceeding. 

Clone the repository and then navigate to the folder using command prompt.
Now type command:
>> python FeedbackScript.py <LMS Username> <LMS Password>

This will start Google Chrome and fill out your feedback automatically. You can change the feedback params from inside the script.
